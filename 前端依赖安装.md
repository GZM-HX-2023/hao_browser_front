# 前端依赖安装指南

## 安装Element Plus图标

前端使用了Element Plus图标，需要安装：

```bash
cd frontend
npm install @element-plus/icons-vue
```

## 完整依赖检查

如果缺少其他依赖，运行：

```bash
npm install
```

## 启动前端

```bash
npm run dev
```

## 访问地址

http://localhost:5173

## 可能缺少的依赖

如果遇到其他导入错误，安装对应的包：

```bash
# Element Plus (应该已安装)
npm install element-plus

# Vue Router (应该已安装)
npm install vue-router

# Pinia (应该已安装)
npm install pinia

# Axios (应该已安装)
npm install axios

# Element Plus 图标 (需要新装)
npm install @element-plus/icons-vue
```

## 验证安装

启动后应该看到：
- ✅ 深色侧边栏
- ✅ "指纹浏览器"品牌名
- ✅ 蓝色"新建浏览器"按钮  
- ✅ 导航菜单
- ✅ 无图标缺失错误
